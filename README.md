# 실시간 예약 웹 서비스 개발 

## 프로젝트 소개
이 프로젝트는 왁싱샵 홈페이지 개발로, 고객이 온라인으로 쉽게 예약하고 서비스를 이용할 수 있도록 지원하는 웹사이트입니다. 
이 홈페이지는 예약 관리, 상담 서비스, 리뷰 및 이벤트 공지사항 등 다양한 기능을 제공하여, 고객에게 편리한 서비스를 제공하고, 관리자에게는 효율적인 운영 도구를 제공합니다.
